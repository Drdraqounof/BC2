# EduPanel Feature List

## Public Pages

### Homepage
- Product overview for teachers and schools
- Messaging focused on moving from data to action
- Feature highlights and campaign examples
- Login entry point to the workspace

### Features Page
- Detailed breakdown of EduPanel capabilities
- Workflow overview from signal to intervention
- Outcome-focused explanation of the product
- Login entry point to the workspace

### Login Page
- Public sign-in screen for teachers
- Entry point into the dashboard workspace

## Dashboard Features

### Active Campaigns
- View all current interventions in one place
- See campaign goals, student groups, and status
- Track campaign progress visually

### Progress Tracking
- Monitor intervention trends across four weeks
- Review key outcome metrics
- Visual progress bars with animation

### Students
- Review students who need support
- See each student's signal and follow-up status
- Open student-specific views from the list

### AI Writer
- Generate outreach and check-in content faster
- Draft teacher and family communication
- Keep campaign context attached to writing workflows

## Core Product Capabilities

### Actionable Insights
- Surface missing work, attendance issues, and risk signals
- Show what needs attention now
- Connect signals to next-step interventions

### Campaign Creation
- Launch a campaign from a classroom issue
- Define a measurable goal
- Select students and timeframe

### Outcome Measurement
- Track whether interventions are working
- Compare progress over time
- Support adjustments based on real results

## Example Campaign Types
- Missing Assignments Recovery
- Attendance Improvement
- Test Preparation Initiative
- Student Performance Boost
- Parent Engagement Push
